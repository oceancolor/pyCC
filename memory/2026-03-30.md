## ARC-AGI-3 核心洞察（2026年3月发布）

- 由 François Chollet / ARC Prize Foundation 发布
- **核心理念**：智能 = 效率（Skill-Acquisition Efficiency），不是"能不能做"而是"用多少资源做到"
- **四大评测柱**：Exploration（探索）、Modeling（建模）、Goal-Setting（目标设定）、Planning & Execution（规划执行）
- **评分方式**：行动效率 = min(1, 人类基准行动数 / Agent实际行动数)
- **当前结果**：所有顶级 LLM（GPT-5/Claude/Gemini）得分 < 1%，人类 100%
- **关键发现**：LLM 推理能力仍依赖训练域知识，无法真正"无指令推断目标"
- ARC-AGI-1 已被 LRM 攻破（90%+），ARC-AGI-2 最高24%，ARC-AGI-3 是新前沿

## Agent Arena 竞技平台 — 设计方案

### 核心创新
- 传统平台：人类写算法代码 → Agent 执行
- 本平台：人类配置 Agent（Skills + MCP + Prompt）→ LLM 实时推理竞技

### 架构
```
Game Engine（状态机/回合）
  ↓ Environment API
Agent Sandbox Layer（隔离）
  ├── Agent A: LLM Core + Skills + MCP
  └── Agent B: LLM Core + Skills + MCP
Match Manager（配对/赛季/Elo积分）
```

### Agent 配置包格式
```yaml
name: "我的探险家"
strategy: rule_based | llm
system_prompt: "..."
skills: [pathfinder, memory_map, opponent_tracker]
mcp_servers: [{name, command}]
```
